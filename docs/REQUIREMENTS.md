# Rapper Tool — 要件定義

## プロダクト概要

日本語ラップ向けの作詞支援ツール。韻候補の取得、AI歌詞生成、韻/フロー分析、メモ帳での手書き、プロジェクト保存を1画面で行う。

---

## 現在の実装状況（2026-05）

### ✅ 完了

| 機能 | 内容 |
|------|------|
| 韻候補取得 | nwnwn / azrhymes / in-note 複合、断片語フィルタ |
| AI歌詞生成 | DeepSeek V4 Pro、ムード/形式/BPM/パンチライン |
| 文章入力 | 「みんなで炊いたガンジャ」等をそのまま歌詞に |
| 分析エンジン | 行末韻・内部韻・母音・モーラ・韻密度スコア |
| 韻品質スコア | 論文 P9-3 参考（5モーラ脚韻・重複ペナルティ・行長） |
| UI | ダークテーマ、タブ（歌詞/韻候補/分析）、日本語UI |
| メモ帳 | 自動保存、Setupと切替、設定保持 |
| メモ帳タグ | `--hook` / `--verse` / `--bridge` 等 |
| メモ帳→分析 | 反映時に自動分析 |
| 歌詞手動編集 | 編集モード + 分析更新ボタン |
| プロジェクト保存 | ローカル（歌詞+韻+Setup+分析+メモ） |
| セットアップ保持 | localStorage、メモ帳往復で消えない |
| スクロールバー | ダークUI統一 |

## ユーザーストーリー

1. **作詞者**: 言葉を入力 → 韻候補を見る → メモ帳で書く → 分析して直す
2. **初心者**: Generate でたたき台 → 手動編集 → 分析で韻密度を確認
3. **継続利用**: プロジェクト保存 → 後日読み込み → 続きから作詞

---

## 画面構成

```
[ヘッダ] ロゴ | ムード/BPM | プロジェクト保存
[左]     セットアップ OR メモ帳
[右]     歌詞 | 韻候補 | 分析
```

---

## API

| エンドポイント | 用途 |
|----------------|------|
| POST /api/rhymes | 韻候補のみ |
| POST /api/generate-lyrics | 韻+生成+分析 |
| POST /api/analyze-lyrics | 歌詞再分析+AI添削 |
| POST /api/save-lyrics | Supabase 保存 |

---

## 非機能要件

- ローカル-first（オフラインでもメモ・設定・プロジェクト保存）
- 生成失敗時はエラーメッセージ表示
- 韻断片（カジャ等）は候補・分析から除外
